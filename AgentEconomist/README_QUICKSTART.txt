📋 快速开始（5分钟）
═══════════════════════════════════════════════════════════

1️⃣ 命令行模式（测试用）
   
   conda activate economist
   cd /root/project/agentsociety-ecosim/AgentEconomist
   python economist_agent.py
   
   输入对话，输入 'exit' 退出

2️⃣ 运行测试脚本

   python test_agent.py           # 基础对话测试
   python test_with_tools.py      # 工具调用测试

3️⃣ 启动 LangGraph Server（用于前端）

   ./start_agent.sh
   # 或
   langgraph dev --port 8123

4️⃣ 启动完整系统（后端 + 前端）

   # 终端 1: 后端
   cd AgentEconomist
   ./start_agent.sh
   
   # 终端 2: 前端
   cd ../frontend
   npm run dev
   
   # 访问 http://localhost:3001
