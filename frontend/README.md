# AgentEconomist Frontend

Next.js frontend for AgentEconomist, built with CopilotKit and LangGraph integration.

## Prerequisites

- Node.js 18+ 
- Python 3.8+
- OpenAI API Key

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

This starts both the UI (port 3001) and agent server (port 8123).

## Available Scripts

- `npm run dev` - Start UI and agent servers
- `npm run dev:ui` - Start only UI server
- `npm run dev:agent` - Start only agent server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
